# Postgraduate Diploma in Cyber Security (E-Learning)

**合作大学**: PSB Academy (新加坡PSB学院)

- **项目简介**：该网络安全研究生文凭课程（线上学习）旨在帮助学生全面理解网络安全领域。在当今数字化威胁日益增加的时代，本课程提供保护系统和数据所需的技能，为学生在这一高需求行业的成功职业生涯做好准备。课程为学生提供网络安全原则和实践的深入理解，以及在道德黑客、计算机取证和信息安全管理等领域的实践技能。学生毕业后，可以通过与顶尖大学的合作关系，有机会升读硕士学位。

- **项目方向与课程结构**：该课程包含6个模块，学生可以按照自己的节奏进行学习。评估方式包括课程作业、在线作业和测验，无笔试。

核心课程：
- Information Security Management (信息安全管理)
- Computer Networking and Network Defence (计算机网络与网络防御)
- Linux Shell Scripting (Linux Shell脚本编程)
- Ethical Hacking and Penetration Testing (道德黑客与渗透测试)
- Computer Forensics Investigations (计算机取证调查)
- Internet of Things: Fundamentals and Security (物联网：基础与安全)

注：模块的授课顺序可能会有变动。

- **项目 FAQ**：
**问：该线上课程有期末考试吗？**
答：没有。该课程为100%线上授课，评估方式仅包括在线测验、测试、作业和在线实验练习，没有传统的笔试。

**问：完成此文凭课程后，是否可以升读硕士学位？**
答：是的。毕业生有机会升读PSB学院合作大学的硕士学位，并可能获得学分豁免，但这需要得到合作大学的最终批准。

**问：如果没有学士学位，可以申请这个课程吗？**
答：可以。该课程为成熟申请者提供了替代入学途径，年满30岁且拥有至少8年工作经验的申请人也符合申请资格。

**问：课程的学习模式是怎样的？**
答：课程采用自定进度、异步的线上学习（Self-paced, Asynchronous E-Learning）模式，学生可以随时随地学习，灵活安排学习时间。
