# Bachelor of Science with Honours in Cyber Security

**合作大学**: Coventry University

- **项目简介**：该网络安全学位项目旨在培养合格的毕业生，他们能够为英国和新加坡增长最快的行业之一做出有效贡献。随着移动用户、数字应用和数据网络的增加，潜在的风险和威胁数量也在增长，因此需要有才华的专业人士来保护系统和数据。该课程使用现实生活中的工具和基于场景的挑战进行培训，通过实践项目和先进的教学方法，为学生提供成为独立学习者的机会。

- **项目方向与课程结构**：核心课程包括：
- Networking (网络)
- Operating Systems (操作系统)
- Information Security (信息安全)
- Problem Solving and Programming (问题解决与编程)
- Cryptography (密码学)
- Computer Forensics (计算机取证)
- Algorithms and Data Structures (算法与数据结构)
- Communication and Collaboration (沟通与协作)
- Secure Programming (安全编程)
- Individual Cyber Security Project (个人网络安全项目)
- Internet of Things (物联网)
- Contemporary Issues in Computing (计算领域的当代问题)

评估方式：评估100%基于课程作业，不设书面考试。评估方式包括作业、项目、测验等。

- **项目 FAQ**：
**问：我持有相关专业的理工学院文凭（Polytechnic Diploma），可以豁免学分或直接进入大二吗？**
答：可以。持有相关信息技术文凭的申请者可豁免模块，最快可在16个月内完成荣誉学士学位。持有本地理工学院相关文凭的申请者，经评估后或可直接入读大三。（基于项目特点推断）

**问：这个项目的考核方式是怎样的？有期末考试吗？**
答：本项目100%基于课程作业进行评估，不设书面考试。考核方式包括作业、项目、测验等。（基于项目特点推断）

**问：如果我没有达到学术入学要求，但有多年工作经验，可以申请吗？**
答：可以。年龄30岁及以上、拥有8年工作经验的成年申请者，其申请将被作为个案进行评估。（基于项目特点推断）
