# Master of Science in Cyber Security (网络安全理学硕士)

**合作大学**: Coventry University (考文垂大学)

- **项目简介**：此网络安全理学硕士项目旨在为学生提供网络安全多个核心领域的知识与必要技能。课程致力于让学生对安全原则有全面而深入的理解，并掌握解决实际安全问题和应对相关挑战的实用技术。

- **项目方向与课程结构**：该项目100%基于课程作业进行考核，无笔试。

核心课程模块包括：
- Network Security (网络安全)
- Ethical Hacking (道德黑客)
- Secure Design and Development (安全设计与开发)
- Cryptography (密码学)
- Intrusion Detection and Response (入侵检测与响应)
- Digital Data Acquisition, Recovery and Analysis (数字数据采集、恢复与分析)
- Security of Emerging Connected Systems (新兴互联系统安全)
- Automotive Cyber Security (汽车网络安全)
- Cyber Security Individual Project (网络安全个人项目)

考核方式：包括作业和项目。
*模块的授课顺序可能会有变动。

- **项目 FAQ**：
**问：此项目是否接受非计算机背景的申请者？**
答：是的。除了计算机科学，持有数学、物理或工程等相关学科的学士学位，或同等水平的专业资格也可申请。此外，项目为年满30岁且拥有至少8年相关工作经验的成熟申请者提供特殊申请通道。（基于入学要求推断）

**问：项目的考核方式是什么，有考试吗？**
答：该项目100%基于课程作业进行考核，没有笔试。评估方式包括作业、项目、测验等。（基于项目亮点推断）

**问：是否可以申请学分/课程豁免？**
答：是的。非全日制学制时长取决于授予的模块豁免。大学也根据规定接受过往学术学习（RPL）和经验学习（RPEL）的认证申请。（基于学制和入学要求推断）
