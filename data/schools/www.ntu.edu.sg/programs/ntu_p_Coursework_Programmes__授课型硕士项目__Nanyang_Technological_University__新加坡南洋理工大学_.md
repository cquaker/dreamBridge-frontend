# Coursework Programmes (授课型硕士项目)

**合作大学**: Nanyang Technological University (新加坡南洋理工大学)

- **项目简介**：南洋理工大学的授课型硕士项目为学生提供了一套规定的学习课程，通常包括若干门课程和一个毕业论文项目。对于大多数项目，学生也可以选择用几门额外的课程来代替毕业论文。学习过程涉及正式课程、讲座、辅导、研讨会、实验工作和书面考试，旨在为学生提供深入的专业知识和实践技能。

- **项目方向与课程结构**：通用课程结构包括核心课程、选修课程和毕业设计/论文。学生遵循一套规定的学习路径，通常毕业论文可以由几门额外的课程代替。具体课程设置因各学院和专业而异，涵盖讲座、辅导课、研讨会、实验项目和书面考试等多种教学形式。

- **学制与开学时间**：学制：待补充（因项目而异）；开学时间：8月、1月、11月（视具体项目而定）

- **申请时间（范围）**：不同学院和项目申请时间不同，主要分为三个入学季：
1. 8月入学（学期制/三学期制）：
   - 大部分项目：11月1日至次年1月底/2月底/3月底不等
   - 李光前医学院项目：1月1日至4月30日或5月28日
2. 11月入学（三学期制）：
   - 主要为应用经济学、区块链、精密科学仪器等项目：3月1日 - 5月31日
3. 1月入学（学期制）：
   - 主要为工程、计算机、人文、公共管理等部分项目：7月1日 - 8月31日
*注：南洋商学院(NBS)和国立教育学院(NIE)有独立的申请流程和时间表。

- **申请流程**：所有申请必须通过学校官方网站在线完成。申请流程如下：
1. 在线填写申请表。
2. 在线通过VISA/Mastercard支付申请费。
3. 在申请门户上传所有必需的电子版证明文件。
4. 提交申请后，可通过申请状态查询入口跟进进度。
*注：在同一入学季申请多个项目需使用同一个申请号。申请费不予退还或转让。

- **申请材料**：
  - 护照/身份证（NRIC）：新加坡永久居民需上传NRIC和再入境许可；其他申请人需上传护照个人信息页。
  - 证件照：近3个月内拍摄的白底彩色证件照。
  - 官方成绩单：由毕业院校出具的包含各科目详细分数的中英文版官方成绩单，需附有评分标准说明。
  - 学位证：中英文版学位证书。
  - 语言能力证明：有效的托福（TOEFL）/雅思（IELTS）成绩单（如适用）。
  - 其他证明文件（如适用）：
  -   - GMAT/GRE成绩单
  -   - 个人简历（Resume）
  -   - 推荐信（仅限申请拉惹勒南国际关系学院(RSIS)和媒体与传播硕士项目）

- **入学要求**：申请人通常需要满足以下基本要求：
- 学术背景：拥有良好的学士学位。 
- 应届毕业生：最后学年在读的本科生可以凭临时学位证明和/或最新的年度考试成绩进行申请。

- **语言要求**：母语非英语的申请人需提供两年内的有效语言成绩。具体要求因学院而异：
- IELTS 7.0 / TOEFL 100：拉惹勒南国际关系学院、人文学院（语言学/创意写作与出版）、黄金辉传播与信息学院
- IELTS 6.5 / TOEFL 100：南洋商学院
- IELTS 6.5 / TOEFL 92：NTUpreneur
- IELTS 6.5（单项不低于6.0） / TOEFL 90：人文学院（翻译与口译）
- IELTS 6.5 / TOEFL 90：艺术、设计与媒体学院
- IELTS 6.5（各单项不低于5.5） / TOEFL 88-89：南洋公共管理中心
- IELTS 6.5 / TOEFL 85：应用经济学硕士
- IELTS 6.0 / TOEFL 85：多数工程学院、理科学院、生物科学学院、应用老年学硕士等
*注：未达到语言要求的申请者也可能被个案考虑，或被要求参加学校安排的面试/测试。

- **作品集/科研要求**：不适用 (视具体项目要求而定，例如艺术、设计与媒体学院的相关专业可能需要)

- **推荐信要求**：仅部分项目要求，例如申请拉惹勒南国际关系学院(RSIS)和媒体与传播硕士项目的申请人需要提交推荐信（Referee Report）。

- **面试**：通常不要求，但对于语言成绩未达标或校方认为有必要的申请者，可能会被要求参加面试或测试。

- **学费**：待补充。学费信息需查询各项目具体页面。除学费外，学生还需考虑生活成本，预估每月开销如下：
- 校内住宿：S$516 - S$756
- 伙食费：S$300 - S$800
- 新加坡境内交通：S$80 - S$200
- 个人开销：S$300 - S$630

其他费用：
- 申请费：待补充（每项课程申请均需缴纳，不可退还）

- **奖学金**：
  - 新加坡公民和永久居民学费补贴：所有入读自费授课型硕士课程的新加坡公民和永久居民可获得S$5,000的补贴。
  - 助学金：有经济援助需求的新加坡公民和永久居民，可获得最高S$15,000的援助。
  - 南大校友学费折扣：所有南大校友在入读非政府补贴的自费课程时，可享受10%的学费折扣。

- **项目 FAQ**：
**问：我可以同时申请多个硕士项目吗？**
答：可以。如果在同一个入学季（例如同为8月入学）申请多个项目，请使用同一个申请号。如果申请不同入学季的项目，则需创建新的申请。

**问：如何查询我的申请状态？**
答：在成功支付申请费的3个工作日后，您可以通过学校官网的申请状态查询入口进行查询。请注意，校方不受理关于申请状态的邮件查询。

**问：如果错过了申请费的支付截止日期会怎样？**
答：申请将被视为无效且不会被处理。申请人需要重新提交新的申请。校方不会对此另行通知。

**问：获得录取后可以申请延期入学吗？**
答：延期入学申请将根据个案情况进行评估，且仅在特殊情况下批准。校方建议，如有延期打算，最好不要接受此次录取，待准备好后再重新申请您感兴趣的项目。

**问：南大是否通过第三方中介机构招生？**
答：不。南洋理工大学未授权任何第三方或代理机构进行招生。所有申请必须由申请人本人直接通过大学官方申请网站提交。
