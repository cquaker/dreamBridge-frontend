# Preparatory Course for International English Language Testing System (IELTS) (雅思国际英语语言测试系统预备课程)

**合作大学**: IELTS (由英国文化协会、IDP教育集团及剑桥大学考试委员会共同管理)

- **项目简介**：该课程由新加坡LSBF学院提供，旨在帮助学生提高英语技能，为雅思考试（学术模块）做全面准备，以达到世界顶尖大学的入学要求。雅思成绩是申请英国、美国、加拿大、澳大利亚等英语国家本科和研究生课程以及向雇主证明英语能力的关键。LSBF是英国文化协会认可的雅思报名合作伙伴，可为学生代办考试报名，并提供IELTS Ready Premium在线资源，包括模拟测试、个性化学习计划等，以确保学生获得最佳备考支持。

- **项目方向与课程结构**：该课程通过分步指导，全面覆盖雅思考试的四个核心模块：

- **IELTS Speaking (雅思口语)**：培养面试中的互动技巧、肢体语言和眼神交流；练习连贯、流畅且语法准确的口语表达；掌握合适的词汇、发音和语调。

- **IELTS Listening (雅思听力)**：训练听取对话和短篇讲座的能力；从图表、地图、时间表等信息中识别关键信息；完成填空、选择题、判断题等多种题型。

- **IELTS Reading (雅思阅读)**：学习使用寻读、略读和扫读等高效阅读技巧；应对多项选择、简答、句子填空等各类题型；理解学术文章结构并处理生词。

- **IELTS Writing (雅思写作)**：掌握雅思写作的规定要求；学习撰写结构清晰、论证有力的学术文章；达到进入高等教育所需的写作熟练度。

- **学制与开学时间**：学制：全日制2个月；非全日制11个课时。
开学时间：每月均有开学。

- **申请时间（范围）**：全年开放申请

- **申请流程**：申请流程根据申请人身份分为本地学生和国际学生两种：

1.  **本地学生**：
    -   步骤1：与教育顾问进行课程前咨询，并提交申请表、最高学历证明、身份证明（NRIC/工作准证）及申请费。
    -   步骤2：招生部门评估申请，成功后学生签署学生合同与咨询说明。
    -   步骤3：支付课程费用。
    -   步骤4：学校出具收据，并为学生购买学费保障计划（FPS）保险。

2.  **国际学生**：
    -   步骤1：进行课程前咨询，并提交申请表、所有学历及成绩单的认证副本、英语水平证明、护照复印件、出生证明、ICA学生准证申请表（Form 16和V36）、照片及申请费等文件进行评估。
    -   步骤2：申请成功后，学生签署录取接受书，并提供财务能力证明等额外文件。
    -   步骤3：学校通过SOLAR+系统提交学生准证申请。批准后，学生需签署学生合同并支付学费。
    -   步骤4：通过电汇、Flywire等方式支付课程费用。
    -   步骤5：学校出具收据，购买FPS保险，学生办理学生准证领取手续。

- **申请材料**：
  - 填写完整的申请表
  - 所有学历资格和成绩单的认证副本（中学起）
  - 英语语言能力证明（如雅思成绩）
  - 学生有效旅行证件（护照）的清晰复印件
  - 经认证的出生证明清晰复印件
  - 马来西亚身份证复印件（如适用）
  - 填写完整的ICA学生准证申请表：Form 16和Form V36
  - 近期护照尺寸照片（白底）
  - 财务能力证明（银行对账单/定期存款等，适用于需要签证国家的学生）
  - 父母和/或配偶的在职证明（如适用）
  - 申请费付款证明

- **项目 FAQ**：
**问：这个课程的学费是否包含官方的雅思考试费用？**
答：文件中未明确说明，但提到学生需要“外部注册（register externally）”参加考试。这通常意味着课程费用不包括官方考试费。LSBF作为英国文化协会的报名合作伙伴，可以协助学生报名考试。（基于项目特点推断）

**问：完成课程后我会获得什么证书？**
答：完成课程后，学生将获得由伦敦商业金融学院（LSBF）颁发的“出席证书（Certificate of Attendance）”。请注意，这不是雅思官方成绩单，后者需要通过参加官方考试获得。

**问：如果我没有任何英语水平证书，可以申请吗？**
答：可以。虽然课程有推荐的语言要求（如雅思5.5），但没有证书的申请人可以在抵达新加坡后参加LSBF的入学分级测试。学校将根据测试结果将你安排到合适的级别。

**问：全日制和非全日制课程有什么区别？**
答：全日制课程为期2个月，每周一至周五上课，每天3小时。非全日制课程总共11个课时，通常在周末（周六或周日）上课，每次4小时，外加2小时的雅思备考测试。
