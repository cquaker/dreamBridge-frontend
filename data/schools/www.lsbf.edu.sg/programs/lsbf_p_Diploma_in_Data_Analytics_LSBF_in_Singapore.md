# Diploma in Data Analytics

**合作大学**: LSBF in Singapore

- **项目简介**：该项目是新加坡LSBF与Amazon Web Services (AWS) 合作开设，旨在为学生提供云计算教育的新机会，以应对IT行业对21世纪高需求工作的要求。通过将AWS的云计算内容整合到数据分析文凭课程中，该项目帮助学生掌握云架构、数据分析、网络安全、机器学习和软件开发等领域的关键技能，为他们进入热门的云计算职业生涯做好准备。学生将有机会使用AWS服务，并将所学知识应用于真实案例中。

- **项目方向与课程结构**：该项目课程采用Amazon Web Services (AWS) 的云计算课程体系，内容将作为数据分析文凭（Diploma in Data Analytics）课程的一部分提供。学生将学习以下领域的技能：
- 云架构
- 数据分析
- 网络安全
- 机器学习
- 软件开发
学生将有机会使用AWS服务（包括机器学习和分析），并将其应用于实际用例中。具体课程模块待补充。
