# Bachelor of Computer Science (Game and Mobile Development)

**合作大学**: University of Wollongong

- **项目简介**：该专业旨在满足当前全球对游戏和多媒体系统开发领域技术毕业生的需求。该专业将使学生掌握在计算机和移动平台（包括手机、平板电脑和可穿戴设备）上开发视频游戏及相关系统的技术技能。学生还将发展传统的计算机科学技能和创造性技能，包括可视化、交互和沟通能力。

- **项目方向与课程结构**：要获得计算机科学（游戏和移动开发）学士学位，学生必须完成或获得豁免总共23个模块，包括核心模块、选修模块和一个毕业设计项目。

**核心模块**：
- Networks and Communications
- Human Computer Interaction
- Fundamental Programming with Python
- Computing and Cyber Security Fundamentals
- System Analysis
- Data Management & Security
- Object Oriented Design & Programming
- Introduction to Web Technology
- IT Project Management
- Software Development Methodologies*
- Mathematics for Computing
- Algorithms & Data Structures
- Advanced Programming
- Database Systems
- Mobile Application Development*
- Interactive Computer Graphics*
- Game Engine Essentials*
- Virtual and Augmented Reality*
- Project
*注：带星号的模块由伍伦贡大学学者授课，并由本地导师支持。

**选修模块**：
学生可从一系列模块中选择，涵盖移动多媒体、电子商务、Web编程、人工智能、网络安全等领域。部分选修课由伍伦贡大学学者授课。

- **项目 FAQ**：
**问：是否可以转学到澳大利亚主校区？**
答：在SIM就读该课程的学生有资格申请转至伍伦贡大学主校区，但这需要通过独立的申请和录取流程。

**问：该项目有哪些学分豁免政策？**
答：学校会根据申请者的学术背景自动评估并授予学分豁免。拥有本地理工学院相关专业文凭的申请者可能获得批量学分豁免。要获得最高学分豁免，申请者的GPA通常需要达到2.0或更高（满分4.0）。

**问：课程的毕业要求是什么？**
答：成功完成课程所有学术要求的学生将被授予澳大利亚伍伦贡大学颁发的计算机科学（游戏和移动开发）学士学位。学生必须修完或被豁免总共23个模块。

**问：课程对出勤率有什么要求？**
答：本地学生的出勤率要求为80%，国际学生的出勤率要求为90%。
