# Bachelor of Computer Science (Digital Systems Security and Cyber Security)

**合作大学**: University of Wollongong (伍伦贡大学)

- **项目简介**：该双专业学士学位课程旨在培养学生在数字安全和数据分析这两个信息通信技术（ICT）核心领域的专业知识和实践技能。课程专为希望拓宽知识和技能基础的学生设计，通过创新的课程结构，使学生能够在与单专业相同的学习时长内，同时攻读数字系统安全和网络安全两个专业，实现双倍的学习影响和经验。课程注重理论基础与计算及应用的实践方法相结合，毕业生将具备应对不断变化的就业市场需求的综合能力。

- **项目方向与课程结构**：该课程要求学生完成总计23个模块（具体数量取决于学分豁免情况），包括核心模块、选修模块和一项毕业设计项目。

核心模块包括：
- Networks and Communications
- Human Computer Interaction
- Fundamental Programming with Python
- Computing and Cyber Security Fundamentals
- System Analysis
- Data Management & Security
- Object Oriented Design & Programming
- Introduction to Web Technology
- IT Project Management
- Mathematics for Computing
- Algorithms & Data Structures
- Advanced Programming
- Database Systems
- Software Development Methodologies*
- System Security*
- Ethical Hacking* (网络安全专业)
- Cybersecurity* (网络安全专业)
- Artificial Intelligence for Cybersecurity* (网络安全专业)
- Cryptography and Secure Applications* (数字系统安全专业)
- Network Security* (数字系统安全专业)
- Web Security* (数字系统安全专业)
- Project
*部分模块由UOW学者授课，并由本地导师支持。

注意：实际修读的模块取决于学生选择的专业组合，且模块设置可能由大学更改，并非所有模块每学期都开设。

- **学制与开学时间**：学制：3年全日制；开学时间：4月、7月

- **申请时间（范围）**：11月10日至1月19日（4月入学）；2月2日至4月13日（7月入学）

- **申请流程**：使用通用申请流程

- **申请材料**：
  - 使用通用申请材料（见通用申请信息），通常包括：
  - 学历证明：GCE 'A' 水准证书、理工学院文凭、成绩单等（非英文文件需提供公证翻译件）
  - 身份证明文件：本地申请人为NRIC，国际申请人为有效护照
  - 护照尺寸照片：近3个月内拍摄的白底电子照片
  - 英语水平证明：雅思（IELTS）、托福（TOEFL）等成绩单（如适用）

- **入学要求**：申请者需满足以下任一资格：
- GCE 'A' 水准：3门H2科目加1门H1科目，总分至少9分（计分方式：A=5, B=4, C=3, D=2, E=1，H1按一半计算）。
- 文凭：新加坡本地理工学院的三年制文凭或同等学历。

学分豁免：
- 拥有相关专业背景（如本地理工学院相关文凭）且GPA不低于2.0（满分4.0）的申请者，可获得最多1.5年的学分豁免。GPA低于2.0的申请者将根据个案进行评估。

- **语言要求**：国际学历申请者需满足以下任一英语语言要求：
- IELTS：总分6.0（单项不低于6.0）
- TOEFL (iBT)：总分70（阅读与写作不低于18分，口语与听力不低于17分）
- Pearson (PTE) Academic：总分54（阅读与写作不低于54分，口语与听力不低于47分）
注意：不接受IELTS Indicator、TOEFL家庭版及PTE Academic Online成绩。

- **项目 FAQ**：
**问：我可以转学到澳大利亚的伍伦贡大学主校区吗？**
答：可以。在SIM就读该课程的学生有资格申请转学至伍伦贡大学校区，但需遵循独立的申请和注册流程。

**问：这个项目提供非全日制（Part-time）学习选项吗？**
答：不提供。根据项目文件，非全日制模式目前不开放申请。

**问：如果我有相关的理工学院文凭，可以申请学分豁免吗？**
答：可以。拥有相关学术背景（如本地理工学院相关文凭）的申请者可能会根据其学业成绩获得学分豁免，最多可豁免1.5年的课程。大学将评估所有已修读的模块以确定豁免资格。

**问：课程由哪些老师授课？**
答：课程由新加坡本地的学者或行业从业者，以及来自澳大利亚伍伦贡大学（UOW）的学者共同授课。

**问：课程的出勤率要求是多少？**
答：本地学生的出勤率要求为80%，国际学生的出勤率要求为90%。
