# Postgraduate Diploma in Data Science and Financial Technology

**合作大学**: University of London (伦敦大学)

- **项目简介**：该创新项目旨在为毕业生提供在金融科技或金融服务领域成功发展所需的定量和分析技能。项目结合了大数据与分析、移动计算和现代金融服务的技术，旨在为企业提供更好的决策支持。学生将通过核心模块学习建立坚实的知识基础和实践经验，并有机会根据个人职业抱负定制学习内容，以应对金融科技对传统银行和金融业带来的变革。

- **项目方向与课程结构**：该项目总计120个学分，学生必须完成8个模块。

课程结构：
- 四个核心模块（共60学分）
- 三个必修模块（共45学分）
- 一个选修模块（共15学分）

核心模块 (Core Modules):
- Financial Data Modelling
- Data Programming in Python
- Statistics and Statistical Data Mining
- Machine Learning

必修模块 (Compulsory Modules):
- Big Data Analysis
- Blockchain Programming
- Financial Markets

选修模块 (Optional Modules, 任选其一):
- Artificial Intelligence
- Data Science Research Topics
- Data Visualisation
- Mathematics for Data Science
- Natural Language Processing
- Neural Networks
- R for Data Science

- **项目 FAQ**：
**问：这个项目是否接受跨专业背景的申请人？**
答：是的。不具备相关学科学士学位的申请人，如果持有任何学科的学士学位，可以通过完成指定的在线预备课程“Foundations of Data Science”后申请（即途径二）。

**问：课程的考核方式是怎样的？**
答：评估由课程作业和部分模块的笔试组成。核心模块、必修模块和选修模块都包含课程作业 I (30%或50%) 和课程作业 II/笔试 (70%或50%)。学生必须在两个评估部分中均达到至少50%的分数才能通过模块。

**问：是否可以豁免（免修）部分课程？**
答：可以。该项目提供学分互认（RPL）。持有新加坡理工学院（Singapore Polytechnic）特定数据科学专业文凭（Specialist Diploma）的毕业生，如果满足相应模块的先修条件，可以申请豁免“Data Programming in Python”和“Data Visualisation”模块。

**问：课程是由哪里的老师授课的？**
答：所有课程均由新加坡管理学院（SIM）的本地教师团队授课。
