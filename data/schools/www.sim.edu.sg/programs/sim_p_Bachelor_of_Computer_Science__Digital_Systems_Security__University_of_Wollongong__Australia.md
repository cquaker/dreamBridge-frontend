# Bachelor of Computer Science (Digital Systems Security)

**合作大学**: University of Wollongong, Australia

- **项目简介**：随着我们的数字生活不断扩展，信息正以前所未有的速度和数量跨越国界和网络。计算机科学（数字系统安全）学士学位课程涵盖了高级安全编程和技术。该课程旨在帮助学生理解信息保护和编码策略，以确保数据的正确性、安全性和真实性。学生将学习如何利用技术的力量，为许多人的生活带来积极的改变。

- **项目方向与课程结构**：该课程共需完成23个模块，包括核心模块、选修模块和1个毕业项目。具体课程如下：

**核心模块:**

- Human Computer Interaction

- System Analysis
- Data Management & Security

- Introduction to Web Technology

- Algorithms & Data Structures

- Database Systems
- Software Development Methodologies
- System Security
- Cryptography and Secure Applications
- Network Security
- Web Security
- Project

**选修模块 (部分):**

注：部分模块由UOW学者授课，并由本地导师支持。模块设置可能由大学更改，并非所有模块每学期都会开设。

- **学制与开学时间**：学制：3年全日制/非全日制；开学时间：4月、7月

- **申请时间（范围）**：11月10日至1月19日（4月入学）；2月2日至4月13日（7月入学）

- **申请流程**：使用通用申请流程

- **申请材料**：
  - 学历证明：需提供GCE 'A' 水准证书、理工学院文凭等（非英文文件需提供公证翻译件）
  - 成绩单：需提供所有学期的官方成绩单（非英文文件需提供公-证翻译件）
  - 身份证明文件：本地申请人为NRIC，国际申请人为有效护照
  - 护照尺寸照片：近3个月内拍摄的白底电子照片
  - 英语水平证明：雅思（IELTS）、托福（TOEFL）等成绩单（如适用）

- **入学要求**：申请者需满足以下学术要求之一：
- GCE 'A' 水准：3门H2科目加1门H1科目，最低总分为9分（计分方式：A=5, B=4, C=3, D=2, E=1，H1科目分数减半计算）。
- 文凭：拥有新加坡本地理工学院的三年制文凭或同等学历。

注：具有相关背景的申请人（如相关理工学院文凭）可能有资格获得学分豁免。GPA不低于2.0（满分4.0）的申请人可获得最高学分豁免。GPA低于2.0的申请将根据具体情况进行评估。

- **语言要求**：国际学历申请者需满足以下任一英语语言要求：
- IELTS (Academic)：总分6.0（单项不低于6.0）
- TOEFL (iBT)：总分70（阅读与写作不低于18分，口语与听力不低于17分）
- Pearson (PTE) Academic：总分54（阅读与写作不低于54分，口语与听力不低于47分）

注：不接受IELTS Indicator, TOEFL (iBT) 家庭版和 Pearson (PTE) Academic Online 成绩。

- **项目 FAQ**：
**问：学生可以转学到伍伦贡大学的澳洲主校区吗？**
答：在SIM就读UOW课程的学生有资格申请转学到伍伦贡校区，但这需要通过独立的申请和注册流程。

**问：课程由哪些老师授课？**
答：该课程由新加坡本地学者和/或行业从业者以及伍伦贡大学(UOW)的学者共同授课。UOW学者授课的模块通常以密集模式进行。

**问：毕业后将获得什么学位证书？**
答：成功完成所有学术要求的学生将获得由澳大利亚伍伦贡大学颁发的计算机科学学士（数字系统安全）学位证书，并可在新加坡参加学位授予仪式。

**问：理工学院毕业生是否可以申请学分豁免？**
答：可以。拥有本地理工学院相关文凭的申请人可能会获得整体学分豁免。大学将评估所有已修读的模块，并根据情况授予适当的豁免。

**问：如果报名人数不足，课程会取消吗？**
答：是的，课程开设有24人的最低人数要求。如果未达到最低人数，该入学批次可能不会开始。学生将在课程开始前一个月内收到通知。
