# Postgraduate Diploma in Data Science (数据科学研究生文凭)

**合作大学**: University of London (伦敦大学)

- **项目简介**：该项目由英国伦敦大学颁发，旨在教授学生如何将技术应用于现实世界的数据科学问题。学生将学习统计学的数学基础、掌握必要的统计技能，并深入了解新兴技术。课程强调通过Hadoop、R等专业软件，在社交媒体、生物医学数据和金融数据等领域进行动手实践。该项目是可堆叠课程，允许学生从研究生证书逐步升读至研究生文凭，并最终攻读硕士学位。

- **项目方向与课程结构**：该项目总计120个学分，学生必须完成8个模块。

课程结构：
- 4个核心模块（60学分）
- 2个必修模块（30学分）
- 2个选修模块（30学分）

核心模块 (Core Modules):
- Big Data Analysis
- Data Programming in Python
- Statistics and Statistical Data Mining
- Machine Learning

必修模块 (Compulsory Modules):
- Data Science Research Topics
- Data Visualisation

选修模块 (Optional Modules) (任选二):
- Artificial Intelligence
- Blockchain Programming
- Financial Data Modelling
- Financial Markets
- Mathematics for Data Science
- Natural Language Processing
- Neural Networks
- R for Data Science

- **项目 FAQ**：
**问：如果我的本科学位不是相关专业，可以申请吗？**
答：可以。申请人若本科学位非相关专业，但学历达到要求，可通过完成在线预备课程 “Foundations of Data Science” 来满足入学途径2的申请条件。

**问：这个项目是否可以豁免学分（Recognition of Prior Learning, RPL）？**
答：可以。如果申请人在过去五年内学习过与本项目模块相似且深度相当的学位级别课程并取得良好成绩，可以申请学分豁免。例如，持有新加坡理工学院特定数据科学专业文凭的毕业生，在满足条件的情况下，可申请豁免 DSM020 Data Programming in Python 和 DSM050 Data Visualisation 模块。

**问：完成这个研究生文凭后可以继续深造吗？**
答：可以。这是一个可堆叠式课程。毕业生在完成该项目后，可以在SIM或全球其他大学继续攻读数据科学理学硕士（MSc in Data Science）学位（需满足目标院校的入学标准）。

**问：课程的考核方式是怎样的？**
答：评估主要由课程作业和笔试（部分模块）组成。通常情况下，每个模块包含两个评估元素，权重可能为30%/70%或50%/50%。学生必须在核心模块的两个评估元素中都至少获得50%的分数才能通过。
