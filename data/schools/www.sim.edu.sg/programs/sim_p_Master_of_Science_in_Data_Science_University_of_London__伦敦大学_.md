# Master of Science in Data Science

**合作大学**: University of London (伦敦大学)

- **项目简介**：该项目由英国伦敦大学颁发学位，并由其成员机构伦敦大学金史密斯学院开发。项目结合了统计学、数据挖掘、机器学习和数据库等成熟领域，以及大数据管理与分析、情感分析和社交网络挖掘等现代前沿领域。学生将学习高效分析海量数据集所需的计算技术，并有机会在Jupyter Notebook等平台上使用Hadoop、R等真实世界的软件。课程旨在为学生在数据挖掘、统计学和机器学习方面打下坚实理论基础，并结合社交媒体、生物医学和金融数据等真实应用。

- **项目方向与课程结构**：该项目总计180学分，学生必须完成10个模块和1个最终项目。

课程结构：
1.  **核心模块 (Core Modules)** - 60学分，必须通过所有评估元素
    - Big Data Analysis (大数据分析)
    - Data Programming in Python (Python数据编程)
    - Statistics and Statistical Data Mining (统计与统计数据挖掘)
    - Machine Learning (机器学习)

2.  **必修模块 (Compulsory Modules)** - 30学分
    - Data Science Research Topics (数据科学研究课题)
    - Data Visualisation (数据可视化)

3.  **选修模块 (Optional Modules)** - 60学分（任选四门）
    - Artificial Intelligence (人工智能)
    - Blockchain Programming (区块链编程)
    - Financial Data Modelling (金融数据建模)
    - Financial Markets (金融市场)
    - Mathematics for Data Science (数据科学数学)
    - Natural Language Processing (自然语言处理)
    - Neural Networks (神经网络)
    - R for Data Science (R语言用于数据科学)
    - Social Media and Network Science (社交媒体与网络科学)

4.  **项目 (Project)** - 30学分
    - Final Project (最终项目)

- **项目 FAQ**：
**问：我之前的学历可以豁免部分课程学分吗？**
答：可以。项目认可先前学习（RPL）。如果申请人过去五年内在学位级别学习过类似深度的模块并取得良好成绩，可以申请学分豁免。例如，持有新加坡理工学院（Singapore Polytechnic）特定数据科学专业文凭的毕业生，如已完成IT8701 Programming for Data Science模块，可申请豁免DSM020 Data Programming in Python模块。

**问：如果我的本科学位不是计算机或数据科学相关专业，可以申请吗？**
答：可以。申请人可以通过“途径二”（Entry Route 2）申请4月入学。前提是申请人拥有受认可的学士学位，并在入学前成功完成指定的在线预备课程“Foundations of Data Science”。

**问：课程的考核方式是怎样的？**
答：评估由课程作业（Coursework）和笔试（Written Examination）组成。核心模块、必修模块和选修模块通常包含两个评估部分，权重分别为30%/70%或50%/50%。最终项目则由70%的课程作业和30%的笔试组成。学生必须在每个评估部分都达到至少50%的分数才能通过模块。

**问：课程由谁授课？**
答：所有课程均在新加坡管理学院（SIM）校园内进行，由SIM的本地教员团队授课。
