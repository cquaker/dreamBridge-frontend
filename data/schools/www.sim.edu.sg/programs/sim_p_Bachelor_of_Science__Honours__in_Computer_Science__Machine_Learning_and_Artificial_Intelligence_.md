# Bachelor of Science (Honours) in Computer Science (Machine Learning and Artificial Intelligence)

**合作大学**: University of London (伦敦大学)

- **项目简介**：该项目由英国伦敦大学授予学位，并由其成员机构伦敦大学金史密斯学院开发。项目专注于机器学习与人工智能领域，旨在通过实践驱动的教学方式，使学生掌握算法和统计模型，构建有效的机器学习系统。课程高度实践化，学生从第一天起就将参与设计、开发和实施软件解决方案，学习包括JavaScript, C++, Node.js和Python在内的多种编程语言。毕业生将具备构建实用人工智能系统、通过智能信号处理让机器理解世界的能力。

- **项目方向与课程结构**：项目学制为三年，每年两个学期。学生每学期最多可修读四个新模块。

**第一年模块:**
学习计算机科学基础，包括编程入门、计算数学和数据结构等。
- CM1005 Introduction to programming I
- CM1010 Introduction to programming II
- CM1015 Computational mathematics
- CM1020 Discrete mathematics
- CM1025 Fundamentals of computer science
- CM1030 How computers work
- CM1035 Algorithms and data structures I
- CM1040 Web development

**第二年模块:**
深化编程技能，探索软件项目开发及图形编程、网络安全等专业课题。

- CM2010 Software design and development
- CM2015 Programming with data
- CM2025 Cyber security
- CM2030 Graphics programming
- CM2035 Algorithms and data structures II
- CM2040 Databases, networks and the web
- CM2045 Professional practice for computer scientists

**第三年模块:**
专注于机器学习和人工智能方向的专业模块，并完成一个最终项目。

- **项目 FAQ**：
**问：这个项目是否提供学分减免或豁免？**
答：是的。项目设有“过往学习认证”（RPL）政策。持有新加坡本地理工学院或SIM颁发的受认可IT相关文凭的毕业生，可能获得最多一年的学分豁免。部分文凭持有者可能需要先完成指定的数学或算法衔接课程才能获得豁免。

**问：课程的教学和考核方式是怎样的？**
答：课程采用混合式学习方法，结合了Coursera平台的在线学习资源和在SIM校园进行的线下实体课。线下课程侧重于案例讨论和实践练习。考核方式通常为课程作业或课程作业与笔试相结合。最终项目则由占80%的课程作业和占20%的相关笔试组成。

**问：如果我的数学或英语成绩不符合直接入学要求怎么办？**
答：对于数学背景不足的申请者，可以参加SIM提供的衔接课程（如Numerical Mathematics、Discrete Mathematics）并考试合格后满足入学要求。对于英语，除了GCE 'O' 水准成绩外，学校也接受多种替代资格，如在三年内考取的雅思（IELTS）或托福（TOEFL）成绩。

**问：毕业生可以去英国参加伦敦大学的毕业典礼吗？**
答：可以。毕业生除了可以参加每年4月在SIM校园举行的毕业典礼外，也可以选择参加每年4月底至5月在英国伦敦举行的伦敦大学毕业典礼。
