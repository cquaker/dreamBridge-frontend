# Web Communication

**合作大学**: Murdoch University

- **项目简介**：该课程结合了网页设计、数字营销和公共关系。学生将学习如何使用一系列数字媒体（包括游戏、移动技术、在线新闻媒体、社交媒体、视频和网站）来设计和制定网络传播活动的策略。课程旨在让学生能够有效地利用Facebook、X、YouTube等网络以及谷歌、雅虎等搜索引擎传达信息和思想，建立设计精良、内容丰富的网站和博客，以实现组织的战略目标。该专业也为学生在社会研究、人文学科和/或创意艺术领域的深造铺平道路。

- **项目方向与课程结构**：学生可以选择攻读单专业或双专业学位。可选的学位课程包括：
- Bachelor of Communication in Web Communication (单专业)
- Bachelor of Communication in Web Communication and Global Media and Communication (双专业)
- Bachelor of Communication in Web Communication and Management (双专业)
- Bachelor of Communication in Web Communication and Marketing (双专业)
- Bachelor of Communication in Web Communication and Strategic Communication (双专业)
- Bachelor of Communication in Web Communication and Tourism and Events (双专业)

- **项目 FAQ**：
**问：我如何才能在16个月内完成这个学士学位课程？**
答：根据文件信息，持有受认可的理工学院文凭或Kaplan文凭的申请人，可能获得最多8至12个课程单元的豁免，从而直接升入大二，有望在16个月内完成学业，但这需要经过大学的最终批准。

**问：在Kaplan新加坡获得的学位与在澳大利亚主校区获得的学位是否相同？**
答：是的。文件明确指出，毕业生将获得与澳大利亚校区毕业生完全相同的学位，并在毕业时获得完整的学位成绩单。

**问：如果我错过了课程，有补救措施吗？**
答：是的，课程提供灵活性。文件提到“提供课堂录像 (Class recording available)”，学生可以按照自己的节奏安排学习，并回顾错过的课程内容。

**问：这个专业是否提供双专业选项？**
答：是的，学生可以选择攻读单专业或双专业。Web Communication可以与Global Media and Communication、Management、Marketing、Strategic Communication以及Tourism and Events等专业结合，攻读双专业学士学位。
