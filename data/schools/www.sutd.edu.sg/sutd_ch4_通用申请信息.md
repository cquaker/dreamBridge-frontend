# 4. 通用申请信息

### 申请流程

SUTD采用全面且分阶段的申请审核流程。无论申请何种学历资格，通用申请流程如下：

1.  **提交申请 (Apply)**：申请人需在规定的申请期限内，通过学校官方网站提交在线申请表格，以申请入学和奖学金。
    *   **申请时段**：
        *   **新加坡-剑桥GCE A-Level**：从A-Level成绩公布之日起至2026年3月19日。
        *   **所有其他资格**：2026年1月2日至2026年3月2日。
    *   **预测成绩申请**：若最终成绩在申请当年的8月前才能获得，申请人可使用最新的学期成绩或预测成绩进行申请。收到录取通知后，需在最终成绩公布后的3天内提交。

2.  **面试 (Conversation)**：初选入围的申请人将收到通知（最晚于4月底），参加与SUTD教职员工或领导的线下面试或在线面试。

3.  **录取结果 (Outcome)**：申请人将在5月中旬前收到电子邮件通知，可登录SUTD招生系统查看申请结果。

4.  **接受录取 (Acceptance)**：获得录取通知的申请人需根据录取通知书中的指引，在规定日期前确认接受录取。

5.  **入学注册 (Matriculation)**：留意关于入学准备和注册流程的电子邮件通知。

6.  **申诉流程 (Appeal)**：未获录取的申请人如有重要的、在申请时未提交的补充信息，可在指定时间内通过招生系统提交申诉。
    *   **常规申诉**：2026年5月8日至2026年5月22日。
    *   **国际文凭（IB）预测成绩申诉**：可在获得最终成绩后，于2026年7月6日至2026年7月10日期间提交申诉。

### 申请材料（通用）

申请人需要通过线上申请系统提交申请，具体的材料清单如下：

*   **在线申请表**：在SUTD招生门户网站上完整填写。
*   **学术成绩单**：包括所有学期的成绩。若最终成绩未出，可提交预测成绩或最新成绩单。
*   **语言能力证明**：对于教学语言非英语的申请者，必须提交相关的英语水平测试成绩。
*   **其他补充材料**：待补充（如个人陈述、推荐信、作品集等，具体要求详见各项目申请指南）。

### 通用入学要求

SUTD采取全面的招生评估方式，除了学术成绩，也重视申请人的个人特质。

*   **学术背景**：
    *   鼓励拥有强大数学和科学（尤其是物理或化学）背景的申请人申请。
    *   对于本地理工学院文凭持有者，工程、计算机、信息技术、人工智能、建筑和科学等相关专业的文凭将被优先考虑。
*   **GPA/成绩要求**：不设最低CGPA或IB分数要求。学校将对申请进行全面评估，综合考量申请人的学术和非学术成就。
*   **个人特质**：学校寻求具备以下核心价值观的学生：
    *   求知欲 (Intellectually curious)
    *   坚韧不拔 (Resilient)
    *   动手能力强 (Hands-on)
    *   敢于冒险 (Risk-taking)
    *   善于协作 (Collaborative)
    *   勇于创新，不循规蹈矩 (Innovative & Nonconforming)
*   **工作经验**：待补充

### 通用语言要求

对于其学历资格的教学语言非英语的申请人，必须提交以下任一英语水平测试的成绩证明：

*   **雅思 (IELTS)**
*   **托福 (TOEFL)**
*   **SAT**
*   **培生学术英语考试 (PTE Academic)**
*   **ACT**
*   **C1 Advanced**

**分数要求**：学校不设最低分数线，语言成绩将作为全面评估的一部分。

### 奖学金信息

SUTD为优秀学生提供多种奖学金，以支持其学业发展。

*   **本科生奖学金**：学校提供由SUTD管理及外界支持的各类奖学金与奖项。申请人可在提交入学申请时一并申请奖学金。具体奖学金类型和条件，待补充。
*   **研究生奖学金 (适用于博士及研究型硕士项目)**：
    *   **SUTD President’s Graduate Fellowship (PGF)**：面向所有国籍的博士生，覆盖学费（最长4年）并发放月度津贴（自2026年1月1日起，新加坡公民为S$5,300，永久居民为S$4,800，国际学生为S$4,300）。
    *   **SUTD PhD Fellowship**：面向所有国籍的博士生，覆盖学费（最长4年）并发放月度津贴（自2026年1月1日起，新加坡公民为S$3,800，永久居民为S$3,400，国际学生为S$3,100）。通过博士资格考试后，津贴每月可增加最高S$500。
    *   **SUTD MEng (Research) Fellowship**：面向所有国籍的研究型硕士生，覆盖学费（最长18个月）并发放月度津贴（自2026年1月1日起，新加坡公民为S$3,800，永久居民为S$3,000，国际学生为S$2,900）。

### 助学金信息

SUTD提供多种助学金和财务援助计划，以确保所有合格学生都有机会接受教育。财务需求评估基于家庭总收入、受抚养家庭成员数量及其他特殊情况。

*   **申请资格**：未获得教育部学费津贴（MOE Tuition Grant）的学生，在整个学习期间将没有资格获得SUTD的财务援助。
*   **申请时间**：新生在被选中参加面试后即可申请财务援助。

**主要助学金与贷款计划**：

| 计划类型 | 适用国籍 | 简介 |
| :--- | :--- | :--- |
| **政府助学金 (Government Bursaries)** | 新加坡公民 (SC) | 根据家庭人均月收入（PCI）提供S$1,350至S$6,300不等的助学金。 |
| **SUTD教育机会助学金 (SEOG)** | 新加坡公民 (SC) | 补充政府助学金，最高可覆盖100%的学费和住宿费，适用于PCI低于S$2,500的学生。 |
| **SUTD管理助学金/学习奖 (Study/Bursary Awards)** | 新加坡公民 (SC), 永久居民 (SPR), 国际学生 (IS) | 金额在S$3,000至S$10,000之间。 |
| **高等教育学费补贴 (TTFS)** | 马来族学生 | 为PCI低于S$2,000的马来族学生提供50%至100%的学费补贴。 |
| **教育部学费贷款 (TFL)** | 新加坡公民 (SC), 永久居民 (SPR) | 最高可贷款新加坡公民应付学费的90%。 |
| **公积金教育贷款计划 (CPFELS)** | 新加坡公民 (SC), 永久居民 (SPR), 国际学生 (IS) | 最高可贷款100%的本科学费。 |
| **教育部学习贷款 (SL)** | 新加坡公民 (SC), 永久居民 (SPR), 国际学生 (IS) | 为符合家庭收入要求的学生提供部分学费贷款和可选的生活津贴（最高S$3,600/年）。 |
